# Python API Client for Digital.ai Release

A Python client library for the Digital.ai Release API, designed to support both modern Python applications and legacy
Jython scripts.

## Important: Jython Compatibility

This library maintains backwards compatibility with Jython scripts by preserving Java naming conventions in API classes.
See [CONTRIBUTING.md](CONTRIBUTING.md) for details on maintaining this compatibility when adding new methods.

## Features

- Full API client for Digital.ai Release
- Supports both Python 3 and Jython 2
- Type hints for better IDE support
- Comprehensive test coverage

## Contributing

When contributing, please read [CONTRIBUTING.md](CONTRIBUTING.md) to understand the naming conventions required for API
compatibility.
